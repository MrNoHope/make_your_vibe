# Make Your Vibe - Android test APK

File tai ve:

- `make_your_vibe_android_arm64_v8a.zip`

Cach cai:

1. Tai file zip ve dien thoai Android.
2. Giai nen file zip.
3. Bam file `app-arm64-v8a-release.apk` de cai.
4. Neu Android hoi quyen, bat `Cho phep cai ung dung khong ro nguon goc`.

Ghi chu:

- Ban nay danh cho da so dien thoai Android moi dung chip `arm64-v8a`.
- Day la ban test, dung de trai nghiem giao dien va tinh nang hien tai.
- Neu dang nhap Google/Facebook khong duoc, kiem tra tai khoan test da duoc them trong Firebase/Facebook Developer.
