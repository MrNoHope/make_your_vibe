# Make Your Vibe - Progress Build

Bản này là Android-only progress build, chia code theo màn hình để dễ chia việc nhóm.

Chưa kèm file nhạc/ambient thật. Khi bổ sung audio, đặt file vào:

- assets/audio/music/
- assets/audio/ambient/

Sau đó khai báo lại assets trong pubspec.yaml nếu muốn phát bằng just_audio setAsset.
